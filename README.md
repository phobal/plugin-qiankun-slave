## Install

```bash
# or yarn
$ npm install plugin-qiankun-slave
```

## Usage

Configure in `.umirc.js`,

```js
export default {
  plugins: [
    ['plugin-qiankun-slave', options],
  ],
}
```

## Options

`name`  子项目名称
